# 开发记录

## [2026-04-17] 项目初始化
- 建立 MyTools 项目目录结构与基础约束。
- 确认技术栈为 .NET Framework 4.8 + WPF。

- 新增 `WireGuardService`，支持连接、断开和状态检测。
- 补充主界面中的 WireGuard 配置与控制区域。

## [2026-04-20] 系统托盘与最小化到托盘
- 引入 `Hardcodet.NotifyIcon.Wpf`。
- 新增托盘图标、右键菜单和窗口恢复/退出逻辑。
- 重写主窗口关闭行为，关闭时默认隐藏到托盘。

## [2026-04-20] 启动闪退修复
- 通过 Windows 事件日志定位到启动期 `System.UriFormatException` / `XamlParseException`。
- 将托盘图标从 `DrawingImage` 改为内置 `AppIcon.png`，修复启动阶段崩溃。
- 去掉 `StartupUri`，改为在 `App.xaml.cs` 中显式创建主窗口。
- 增加 UI 线程、`AppDomain`、`Task` 级别的全局异常处理。
- 启动失败时写入 `MyTools.startup.log`，并弹窗提示日志位置。

## [2026-04-23] 区域截图
- 新增 `ScreenshotService` 与 `RegionPickerForm`，支持全屏拖选矩形区域并保存 PNG 到 `Screenshots`。
- 主界面侧栏增加「截图」入口与说明页；截图时临时隐藏主窗口，完成后提示保存路径。
