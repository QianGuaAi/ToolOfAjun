# 开发记录
## [2026-04-17] 项目初始化
- 建立 MyTools 项目目录结构与基础约束。
- 确认技术栈为 .NET Framework 4.8 + WPF。
## [2026-04-17] 单文件发布配置
- 引入 `Costura.Fody`，用于 Release 产物单文件打包。
- 在项目中加入 SQLite 原生库嵌入配置。
## [2026-04-17] WireGuard 功能接入
- 新增 `WireGuardService`，支持连接、断开和状态检测。
- 补充主界面中的 WireGuard 配置与控制区域。
## [2026-04-20] 系统托盘与最小化到托盘
- 引入 `Hardcodet.NotifyIcon.Wpf`。
- 新增托盘图标、右键菜单和窗口恢复/退出逻辑。
- 重写主窗口关闭行为，关闭时默认隐藏到托盘。
## [2026-04-20] 启动闪退修复
- 通过 Windows 事件日志定位到启动期 `System.UriFormatException` / `XamlParseException`。
- 将托盘图标从 `DrawingImage` 改为内置 `AppIcon.png`，修复启动阶段崩溃。
- 去掉 `StartupUri`，改为在 `App.xaml.cs` 中显式创建主窗口。
- 增加 UI 线程、`AppDomain`、`Task` 级别的全局异常处理。
- 启动失败时写入 `MyTools.startup.log`，并弹窗提示日志位置。
## [2026-04-23] SQL Server 表导出 Excel
- 新增 SQL 导出模块，支持输入服务器、端口、用户名和密码测试连接。
- 连接成功后异步加载数据库列表与用户表列表，按 `schema.table` 展示。
- 新增整表导出 `.xlsx` 能力，并在导出前校验 Excel 单工作表最大行数限制。
- 新增应用日志服务，记录 SQL 导出关键步骤，同时避免记录密码和完整连接字符串。
## [2026-04-23] SQL Server 旧版本兼容修复
- 修复部分旧版 SQL Server 返回“对象名 sys.databases 无效”导致测试连接后无法加载数据库列表的问题。
- 数据库列表读取保留 `sys.databases` 优先路径，失败时自动降级到 `sp_databases`，提升 SQL Server 2000/兼容服务端可用性。
## [2026-04-23] SQL 导出交互增强
- 数据表下拉框改为可输入筛选，按 `schema.table` 文本进行本地实时过滤。
- 新增 SQL 连接历史记录，服务器地址、用户名和密码可从历史下拉选择，并在启动时回填最后一次成功连接信息。
- 历史密码使用 Windows DPAPI 按当前用户加密保存，避免明文写入本地配置文件。
## [2026-04-23] 图标与密码历史下拉修正
- 将密码历史选择从嵌入式 ComboBox 改为右侧图标按钮加遮罩菜单，修复下拉边框显示异常。
- 重新设计现代科技感应用图标，并将 `Resources\AppIcon.ico` 配置为程序图标。
## [2026-04-23] 图标配色调整
- 将应用图标调整为红蓝科技风格，加入星条旗暗纹、星形节点和红色能量线，减少大面积蓝色占比。
